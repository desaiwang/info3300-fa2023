    typeof(null) returns object even though null is a canonical JavaScript type 
    because of design decisions in early Javascript versions.
    

    Type coercion is when an object's type is changed into another type automatically
    to avoid errors. (JavaScript is intentionally designed to avoid crashing.)
    Examples of unusual type coercions:
    "1" + 2 = "12" even though 1+2 =3 because 2 is converted to a string,
    and "1" concatenated with "2" is "12"
    2 + !null = 3 because !null evaluates to true and is coerced
    to become a number, 1, and 2+1=3.
